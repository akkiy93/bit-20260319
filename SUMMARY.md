## うまくいかなかったこと ⚠️

### PPTXファイルの生成
- MarpのPPTX出力にはブラウザ（Chromium）が必要
- WSL2環境からWindowsのChromeをPuppeteer経由で起動するのが不安定で断念
- VS Code Marp拡張でも同様のエラーが発生
- **代替策として** GitHub Pages（HTML公開）で運用することにした

### Google SlidesへのPDF取り込み
- Google SlidesはPDFの直接インポートに非対応
- Google Drive経由で開く方法があるが今回は未試行

## 今日の作業を通じて見えた「半自動」の価値

| 作業 | 人間がやった部分 | AIがやった部分 |
|------|---------------|--------------|
| テスト仕様書 | 初期項目の提示・レビュー・判断 | 仕様書との照合・追加提案・Markdown整形 |
| Excel変換 | 「こういうフォーマットがほしい」の指示 | スクリプト作成・実行 |
| スライド作成 | 構成の方向性・文言の修正指示 | 初稿作成・再生成 |
| GitHub公開 | リポジトリ名・公開範囲の判断 | リポジトリ作成・ファイルプッシュ・Pages設定 |

→ 人間は「何を作るか・これでいいか」の判断だけに集中できた。
